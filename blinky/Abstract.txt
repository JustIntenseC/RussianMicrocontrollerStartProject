Проект "blinky" - Простой пример с моргающим светодиодом для микроконтроллера K1921VK035 (НИИЭТ)

Параметры проекта:
   - XTAL   	 =  16.00 МГц
   - SYSCLK(PLL) = 100.00 МГц
   - UART1(Rx - PB.9, Tx - PB.8) BaudRate = 115200

Проект адаптирован для работы с платами: NIIET-BB-K1921VK035 и NIIET-DEV-K1921VK035.
Для сборки проекта под определенную плату, достаточно добавить в проект соответствующий каталог из "blinky->bsp"

Для компиляции проекта необходимо определить в настройках компилятора (вкладка "С/С++", Define) 3 значения:
RETARGET,
SYSCLK_PLL,
OSECLK_VAL=16000000

RETARGET - для переопределения функции printf на передачу по UART1
SYSCLK_PLL - выбор источника системной частоты (SYSCLK_PLL, SYSCLK_OSI, SYSCLK_OSE)
OSECLK_VAL=16000000 - ввод значения частоты внешнего тактового сигнала в Гц.

Структура проекта:

blinky:
   | 
   |-> CMSIS - Каталог
   |	|-> Core    - Каталог, содержащий стандартные файлы описания Arm Cortex-M4 
   |	|-> Device  - Каталог, содержащий файлы устройства K1921VK035: header, startup, system
   |
   |-> bsp - Каталог, содержащий файлы поддержки отладочных плат (Board Support Package)
   |    |-> NIIET-BB-K1921VK035   - Каталог, содержащий определения для демо платы (BreadBoard совместимой) "NIIET-BB-K1921VK035" 
   |    |    |-> bsp.c
   |    |    |-> bsp.h
   |    |
   |    |-> NIIET-DEV-K1921VK035  - Каталог, содержащий определения для платы макетно-отладочной "NIIET-DEV-K1921VK035" 
   |	     |-> bsp.c
   |	     |-> bsp.h
   |
   |-> Retarget  - Каталог, содержащий файлы переопределения функции printf на передачу по UART1
   |	|-> Core    - Каталог, содержащий стандартные файлы описания Arm Cortex-M4 
   |	|-> Device  - Каталог, содержащий файлы устройства K1921VK035: header, startup, system
   |
   |-> main.c    - исходный код программы
